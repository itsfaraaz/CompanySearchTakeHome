# Frontend Setup

This frontend uses:
- **React** - UI library
- **TypeScript** - Type-safe JavaScript
- **Vite** - Fast build tool
- **Material-UI** - UI component library
- **Tailwind CSS** - Utility-first CSS framework

## Setup

1. Ensure you have Node.js v18+ or v20+ installed
   ([Install Node.js](https://nodejs.org/en/download/package-manager/all) if you don't have it yet)

2. Ensure you have Yarn installed
   ([Install Yarn](https://yarnpkg.com/getting-started/install) if you don't have it yet)

3. Make sure you are in the frontend directory:
   ```bash
   pwd
   # Should show: .../frontend
   ```

4. Install dependencies:
   ```bash
   yarn install
   ```

5. Start the development server:
   ```bash
   yarn dev
   ```

6. Open http://localhost:5173 in your browser to view the app

## Available Scripts

- `yarn dev` - Start the development server
- `yarn build` - Build for production
- `yarn lint` - Run ESLint
- `yarn preview` - Preview production build locally
