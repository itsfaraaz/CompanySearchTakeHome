# Fullstack Take-Home Project Template

This repository contains the scaffolding for a fullstack take-home project.

## Structure

This repo contains 2 services:

1. **Backend** - Dockerized Python backend with FastAPI and PostgreSQL database
2. **Frontend** - React/TypeScript frontend built with Vite
